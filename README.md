# RedBloom-Effect

A playful interactive web experience where your hand gestures control a blooming flower in real time using your webcam and MediaPipe hand tracking.

## Live Preview

Open the project here:
https://htmlpreview.github.io/?https://raw.githubusercontent.com/mitulaghara/RedBloom-Effect/main/index.html

## Features

- Real-time hand tracking with MediaPipe
- Left hand pinch controls bloom intensity
- Right hand pinch controls growth
- Hand movement creates a wind effect
- Glowing animated flower rendered on a canvas overlay

## How to Run Locally

1. Clone the repository
2. Open the project folder in a browser
3. Allow camera access when prompted

> For the best experience, use a modern browser such as Chrome or Edge.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- Canvas API
